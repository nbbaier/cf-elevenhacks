# Impeccable Design Context

## Design Context

### Users
Focus workers who use ambient sound to maintain calm, distraction-free concentration. They interact with the app during active work sessions — they want to set a soundscape quickly and get out of the way. The interface should require minimal attention once configured. Emotional goal: calm, quiet confidence. The UI should feel like the sonic environment it produces: unhurried and comfortable.

### Brand Personality
**Warm · Organic · Textured**

Voice is soft and unhurried — not clinical, not flashy. Copy is minimal. Interactions feel natural, not mechanical. Visual warmth comes from the color system (purple leaning toward red-violet rather than cold blue-purple), subtle surface texture via light borders and layering, and organic use of rounded shapes.

### Aesthetic Direction
- **Light-first**: Light mode is the primary experience. White backgrounds, generous whitespace, subtle borders. Dark mode should be equally polished but secondary.
- **Color used sparingly**: The purple primary (`oklch(0.45 0.2 275)`) should feel like a quiet accent, not a dominant force. Use it to signal action and focus states only.
- **Airy and spacious**: Cards and components breathe. Don't fill space just because it's available.
- **Soft depth**: Use light shadows (`shadow-sm`, `shadow-md`) and subtle rings (`ring-1 ring-foreground/5`) to suggest layers without heaviness.
- **Layer colors**: The 8-hue OKLCH layer color system is a key brand moment — colorful but perceptually balanced, not jarring.
- **Typography**: Inter Variable only. Weight contrast (semibold headings, regular body) creates hierarchy without needing size extremes.

### Anti-patterns to Avoid
- **Glassmorphism / frosted glass** — no blur-backdrop, translucent panels, or frosted card effects
- **Neon / cyberpunk glows** — no `drop-shadow` or `box-shadow` glow effects in accent colors
- **Heavy gradients** — no large-fill background gradients or gradient overlays on cards
- **Skeuomorphic elements** — no textures mimicking real materials, no fake knobs or physical controls
- **Over-playfulness** — this is a tool, not a toy; avoid cartoon roundness or illustration-heavy layouts

### Design Principles

1. **Calm before clever** — If an interaction adds visual noise, cut it. The UI should feel quieter than you expect.
2. **Warmth without weight** — Use color and roundness for warmth, but keep surfaces light and uncluttered. Nothing should feel heavy.
3. **The audio is the hero** — UI chrome should fade into the background once a soundscape is playing. Controls should feel subordinate to the experience they enable.
4. **Deliberate restraint on color** — The layer color system is a feature; everywhere else, desaturate. One source of color vibrancy at a time.
5. **Organic, not decorative** — Rounded shapes, soft shadows, and Inter's humanist letterforms provide all the texture needed. Don't add decoration for its own sake.

### Technical Notes
- Color system: OKLCh throughout (`oklch(L C H)`) — maintain perceptual uniformity when adding new colors
- Border radius base: `0.625rem` — components use `rounded-2xl` through `rounded-4xl`
- Spacing: Tailwind defaults, prefer `gap-6` for component interiors, `gap-3`–`gap-4` for tighter groups
- Icons: Phosphor Icons only
- Font: Inter Variable via `@fontsource-variable/inter`
- Max content width: `max-w-2xl` (640px)
